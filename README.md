# SOME-TALES

这是星闻道、椰椰椰椰和其他的朋友们用来堆砌TRPG战役设定和独立游戏设定的一个网站。  
我们欢迎任何有着架空世界爱好的朋友们的加入，也期待大家能够将自己创造的世界通过作品的形式展现出来。  

*“也许你心中所幻想的那个世界真的会在某个你所无法窥见的时空中存在。”*
